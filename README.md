# poszukiwanie-pierwiastkow-funkcji-GUI

Data projektu: 02-2021

Aplikacja pozwala na podanie ścieżki wejścia, dla którego zwizualizuje w GUI - przebieg funkcji wraz z jej pierwiastkami, rozwiązaną wg metody poszukiwania pierwiastków funkcji zadeklarowanej w postaci klasy w odpowiednim folderze, program można rozszerzyć o nowe metody dziedziczące po klasie abstrakcyjnej. Użyto metody Newtona.
